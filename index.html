Intro

Installable via npm install --save multihashing-async, it can also be used directly in the browser.

Quick Example

'use strict'

const multihashing = require('multihashing-async')
const buf = Buffer.from('beep boop')

function print (err, mh) {
  if (err) {
    throw err
  }
  // eslint-disable-next-line
  console.log(mh)
}
multihashing(buf, 'sha1', print)
// => <Buffer 11 14 7c 83 57 57 7f 51 d4 f0 a8 d3 93 aa 1a aa fb 28 86 3d 94 21>

multihashing(buf, 'sha2-256', print)
// => <Buffer 12 20 90 ea 68 8e 27 5d 58 05 67 32 50 32 49 2b 59 7b c7 72 21 c6 24 93 e7 63 30 b8 5d dd a1 91 ef 7c>

multihashing(buf, 'sha2-512', print)
// => <Buffer 13 40 14 f3 01 f3 1b e2 43 f3 4c 56 68 93 78 83 77 1f a3 81 00 2f 1a aa 5f 31 b3 f7 8e 50 0b 66 ff 2f 4f 8e a5 e3 c9 f5 a6 1b d0 73 e2 45 2c 48 04 84 b0 ...>

There are more available, so take a look at the docs below for a full list. This documentation aims to be comprehensive, so if you feel anything is missing please create a GitHub issue for it.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save multihashing-async

You can then require() multihashing-async as normal:

const multihashingAsync = require('multihashing-async')

In the Browser

multihashing-async should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of multihashing-async, including index.js and index.min.js, are included in the /dist folder. multihashing-async can also be found on unpkg.com under

  • https://unpkg.com/multihashing-async/dist/index.min.js
  • https://unpkg.com/multihashing-async/dist/index.js
